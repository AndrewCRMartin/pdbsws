1. Assumes you have the following installed:
   a) PostgreSQL
   b) FASTA
2. Assumes you have created a PostgreSQL database called pdbsws
3. Run ./createdb.sh
4. Examine build.sh and ensure you have the required data:
   a) UniProt .dat and .fasta files
   b) PDB files
5. Modify build.sh to alter paths etc as required.
6. Run ./build.sh
